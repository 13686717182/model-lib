# 概述
适用于手机型号查询的数据库

# 存储格式
model:name:brand

# 在线 API
正常状态返回 0，name 为手机名称，brand 为手机品牌
```
https://model-lib.4kb.cn/api/model/{model}
```
example:
```
https://model-lib.4kb.cn/api/model/SM901
```
